sed
	Stream EDitor,行编辑器，流编辑器
		一次处理一行内容
		把当前处理的行存储在临时缓冲区中，称为“模式空间”
		处理完成后，把缓冲区的内容送往屏幕。接着处理下一行
		同一行中只处理给定的字符串或模式匹配的内容，没给出的内容依旧打印处理
		文件内容并没有改变，除非使用特定的选项
	sed [option]... 'script' inputfile...
	[option]:
		-n：使用安静(silent)模式。在一般 sed 的用法中，所有来自STDIN的数据一般都会被列出到终端上。但如果加上-n参数后，则只有经过sed特殊处理的那一行(或者动作)才会被列出来
		-e：进行多项编辑，即对输入行应用多条sed命令时使用
		-f：/PATH/TO/SCRIPT_FILE ：从指定文件中读取编辑脚本
		-r：支持使用扩展正则表达式
		-i：修改原文件，而不是输出到终端。
			i.xxx 把修改的原文件备份成file.xxx 再去修改file文件

	'script'：'地址命令'
		地址定界：
			定址用于决定对哪些行进行编辑。地址的形式可以是数字、正则表达式、或二者的结合。
			(1)不给地址：
				对全文进行处理
			(2)单地址：
				#：指定的行
				/pattern/：被此处模式所能够匹配到的每一行
			(3)地址范围：
				#,#：第一个#行到第二个#行
				#,+#：#行到偏移第一个#行的+#行
				/pat1/,/pat2/：第一次模式/pat1/匹配到的行，到第一次模式/pat2/匹配到的行，支持grep类型的正则表达式
				#,/pat1/：#行到第一次模式/pat1/匹配到的行
			(4) ~ ：步进
				1~2  奇数行：第1行，以后每次加2行，奇数行
				2~2  偶数行：第2行，以后每次加2行，偶数行

		编辑命令：（多个命令之间用；隔开）
			d：删除模式空间匹配的行
			p：显示模式空间中的内容，与-n结合使用，可以显示指定的行
			a \text：在行后面追加文本；支持使用\n(换行)实现多行追加
			i \text：在行前面插入文本；支持使用\n(换行)实现多行插入
			c \text：替换行为单行或多行文本
			w /path/to/somefile：保存模式匹配的行至指定文件
			r /path/from/somefile：读取指定文件的文本至模式空间中匹配到的行后
			=：为模式空间匹配到的行打印行号
			!:模式空间中匹配行取反处理，比如!d：前面的行不删除,!p匹配的行不显示
				!d== -n p

		查找替换:
			s/string1/string2/[替换标记]
				用string2中的字符串替换string1中的字符串
				string1可以为模式，string2可以引用string1中的字符串，但是不能使用模式
			支持使用其它分隔符s@@@ ，s###
			替换标记：
				g：行内所有被字符串/模式匹配的字符串全部替换（全局替换）
				w：/PATH/TO/SOMEFILE ：将替换成功的行保存至文件中
			&：引用string1的字符串（&写在string2字符串中的）

练习：处理/etc/fstab路径, 使用sed命令取出其目录名和基名处理/etc/fstab路径, 使用sed命令取出其目录名和基名
	~]# echo "/etc/fstab" | sed -r 's@(.*)/([^/]+)/?$@\1@'
	~]# echo "/etc/fstab" | sed -r 's@(.*)/([^/]+)/?$@\2@'

在/etc/fstab 文件中不以#开头的行的行首增加#号
	~]# sed 's@^[^#]@#&@' /etc/fstab

利用sed取出ifconfig 命令中本机的IPv4 地址
	~]# ifconfig | sed -n "/inet.*Bcast/p" | sed -r 's@.*addr:(.*)\<Bcast.*@\1@'


	高级编辑命令：
		h：把模式空间中的内容覆盖至保持空间中
		H：把模式空间中的内容追加至保持空间中
		g：从保持空间取出数据覆盖至模式空间
		G：从保持空间取出内容追加至模式空间
		x：把模式空间中的内容与保持空间中的内容进行互换
		n：读取匹配到的行的下一行覆盖至模式空间
		N：追加匹配到的行的下一行至模式空间
		d：删除模式空间中的行
		D：删除当前模式空间开端至\n的内容（不在传至标准输出），放弃之后的命令，但是对剩余模式空间重新执行sed

练习：
	sed -n 'n;p' FILE
	只显示偶数行
	sed '1!G;h;$!d' FILE
	倒序显示
	sed '$!N;$!D' FILE
	只文件最后2行
	sed '$!d' FILE
	只文件最后1行
	sed 'G' FILE
	在文件的每一行下面插1个空行
	sed 'g' FILE
	在文件所有行置换成空行
	sed '/^$/d;G' FILE
	把文件的每一行下面是否有无多个空行都变成1个空行
	sed 'n;d' FILE
	只显示奇数行
	sed -n '1!G;h;$p' FILE
	倒序显示
